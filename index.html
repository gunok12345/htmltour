สอน HTML อย่างง่าย

HTML (HyperText Markup Language) คือ ภาษาหลักสำหรับสร้างเว็บไซต์
โครงสร้างพื้นฐานของ HTML มีดังนี้

ตัวอย่างโครงสร้าง HTML

<!DOCTYPE html>
<html>
  <head>
    <title>ชื่อเว็บของคุณ</title>
  </head>
  <body>
    <h1>สวัสดี HTML!</h1>
    <p>นี่คือตัวอย่างย่อหน้า</p>
  </body>
</html>

องค์ประกอบสำคัญ

  • <html> : ครอบทั้งหมดของหน้าเว็บ
  • <head> : ส่วนหัว (ข้อมูล meta, title, link)
  • <body> : เนื้อหาหลักที่แสดงบนเว็บ

ลองแก้ไขและทดลองเอง!

ทดลอง HTML ออนไลน์

แท็ก HTML พื้นฐานที่ควรรู้

Tag คำอธิบาย
<!DOCTYPE>ประกาศชนิดเอกสาร HTML
<html>ครอบทั้งหมดของหน้าเว็บ
<head>ส่วนหัว (meta, title, link, script)
<title>ชื่อเรื่องของเว็บ
<meta>ข้อมูล meta เช่น charset, viewport
<link>เชื่อมโยงไฟล์ภายนอก เช่น CSS
<body>เนื้อหาหลักของเว็บ
<h1> - <h6>หัวข้อ (ขนาดใหญ่ไปเล็ก)
<p>ย่อหน้า
<a>ลิงก์
<img>แทรกรูปภาพ
<ul>, <ol>, <li>รายการ (ไม่เรียง/เรียงลำดับ)
<table>, <tr>, <td>, <th>ตาราง
<div>กล่องแบ่งส่วน
<span>กล่องอินไลน์
<br>ขึ้นบรรทัดใหม่
<hr>เส้นคั่น
<form>, <input>, <button>, <label>ฟอร์มและปุ่ม
<script>ฝัง JavaScript
<style>ฝัง CSS
