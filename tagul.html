การทำงานของแท็ก <ul>, <ol>, <li>

<ul> ใช้สร้างรายการแบบไม่เรียงลำดับ (จุดนำหน้า)
<ol> ใช้สร้างรายการแบบเรียงลำดับ (ตัวเลขนำหน้า)
<li> ใช้สำหรับแต่ละรายการใน <ul> หรือ <ol>

ตัวอย่างการแสดงผล

รายการแบบไม่เรียงลำดับ (ul):
  • HTML
  • CSS
  • JavaScript
รายการแบบเรียงลำดับ (ol):
  1. เตรียมไฟล์
  2. เขียนโค้ด
  3. ทดสอบ

ตัวอย่างโค้ด

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

<ol>
  <li>เตรียมไฟล์</li>
  <li>เขียนโค้ด</li>
  <li>ทดสอบ</li>
</ol>
หมายเหตุ: สามารถซ้อน <ul> หรือ <ol> ได้หลายระดับเพื่อสร้างรายการย่อย
</ul> หรือ </ol>
  • HTML
  • CSS
  • JavaScript
