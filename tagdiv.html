การทำงานของแท็ก <div>

แท็ก <div> ใช้สำหรับแบ่งส่วนหรือกลุ่มเนื้อหาใน HTML สามารถนำไปจัดรูปแบบด้วย CSS ได้อย่างอิสระ

ตัวอย่างการแสดงผล

นี่คือกล่องที่สร้างด้วย <div> และตกแต่งด้วย CSS

ตัวอย่างโค้ด

<div style="background:#2d6cdf;color:#fff;padding:24px;border-radius:10px;text-align:center;">
    นี่คือกล่องที่สร้างด้วย <div>
</div>
หมายเหตุ: <div> ไม่มีผลต่อการแสดงผลโดยตรง แต่ใช้จัดกลุ่มและตกแต่งเนื้อหา
